# VPet插件: 更多的TTS(MoreTTS)

## 介绍

平台为Steam虚拟桌宠模拟器(VPet_Simulator)

本插件依托于 https://huggingface.co/spaces/ikechan8370/vits-uma-genshin-honkai ，使用了其中的TTS模型，侵权删。
